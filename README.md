1. v1.0 - просто слайдер, который передвигается по кругу. Элементы пологнаны по ширине контейнера и изменяют свой размер при изменении ширины контейнера.

2. v2.0 - если задать в верстке элементу фиксированную ширину, можно центрировать элементы относительно указанного в объекте настройки.

3. v3.0 - добавлены точки, которые указывают на позицию центрального элемента (или первого, если слайдер не центрирован).

Инициализация слайдера в основном скрипте:
------------------------------------------

import Slider from './simpleSlider';

const data = {
    // обязательный параметр - обертка слайдера, внутри которой находится трек с элементами
    wrapper: '.slider',
    // стрелки для движения влево-вправо. Если они не указаны, то включится режим "swipe"
    prev: '.slider__prev',
    next: '.slider__next',
    // сколько секунд будет проходить анимация движения слайдера (по умолчанию 0.5)
    interval: .8,
    // только при фиксированной ширине элемента можно центрировать слайдер по номеру 
    centralItem: 3, 
    // если хотим организовать свайп слайдера
    swipe: true,    
    // если есть сверстанные точки 
    dots: {
            panelClass: 'gallery-dot',  // обертка точек
            activeClass: 'dot-active'    // активный класс точки
        }
          
}

const carusel1 = new Slider(data);
